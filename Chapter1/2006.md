## 2006 求奇数的乘积

### 一、题目

#### 问题描述

> 给你n个整数，求他们中所有奇数的乘积。

#### 输入数据

> 输入数据包含多个测试实例，每个测试实例占一行，每行的第一个数为n，表示本组数据一共有n个，接着是n个整数，你可以假设每组数据必定至少存在一个奇数。

#### 输出数据

> 输出每组数中的所有奇数的乘积，对于测试实例，输出一行。

#### 输入样例

> 3 1 2 3 
> 4 2 3 4 5

#### 输出样例

> 3
> 15

#### 题目来源

> HDU 2006 http://acm.hdu.edu.cn/showproblem.php?pid=2006

### 二、题解

#### 解题思路

题目思路并不复杂，可使用循环，并判定，若输入为偶数则跳过，输入为奇数则相乘。

#### 参考程序

```
#include <stdio.h>
int main()
{
	int n, b, m;
	while (scanf("%d", &n) != EOF)
	{
		m = 1;
		while (n--)
		{
			scanf("%d", &b);
			if (b%2==1)
			{
				m *= b;
			}
		}
		printf("%d\n", m);
	}
	return 0;
}
```
