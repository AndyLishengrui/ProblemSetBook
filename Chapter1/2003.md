## 2003 求绝对值

### 一、题目

#### 问题描述

> 求实数的绝对值。

#### 输入数据

> 输入数据有多组，每组占一行，每行包含一个实数。

#### 输出数据

>  对于每组输入数据，输出它的绝对值，要求每组数据输出一行，结果保留两位小数。

#### 输入样例

> 123 
> -234.00

#### 输出样例

> 123.00
> 234.00

#### 题目来源

> HDU 2003 http://acm.hdu.edu.cn/showproblem.php?pid=2003

### 二、题解

#### 解题思路

题目非常简单，自行写判断或者用fabs()函数均可。注意输入数据有多组，并注意输出格式。

#### 参考程序

```
#include<stdio.h>
#include<stdlib.h>
#include<math.h> 
int main()
{
double a;
while(scanf("%lf",&a)!=EOF){
	printf("%.2lf\n",fabs(a));
}
return 0;	
}
```
